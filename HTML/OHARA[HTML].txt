● <input type="text" name="first name" id="user name" value="Bharath" readonly="readonly" size="50" >
type="text":
    Specifies that the input field should accept free-form text input.

name="first name":
    Defines the name of the input field. This name is used to identify the input when the form is submitted, and it is often used in server-side scripts to process the form data.

id="user name":
    Provides a unique identifier for the input field. The ID is used to associate the label with the input field and can be referenced in CSS styles and JavaScript.

value="Bharath":
    Sets the initial value of the input field to "Bharath". This value will be displayed in the input field when the page loads. However, since the readonly attribute is present, the user cannot modify this value.

readonly="readonly":
    Makes the input field read-only, meaning that the user cannot edit or modify the content. The value specified in the "value" attribute will be displayed, but the user won't be able to change it.

size="50":
    Specifies the visible width of the input field in characters. In this case, the input field will be 50 characters wide when displayed on the webpage.
